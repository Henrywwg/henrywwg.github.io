Ongoing

This PWM controller is an ongoing project to create a remotely controlled PWM driver capable of supporting multiple high wattage devices. The device is digitally controlled and will allow up to 5 devices to be controlled each with their own PWM signal. Additionally, the controller will be able to be programmed remotely and individual PWM configurations can be saved and sent between devices with the end goal of allowing any number of people to emulate the same PWM experience.

AXI4-Stream Hardware Downscaling

As part of finishing up my B.S. in Computer Engineering, I did a Senior Design/Capstone project (ECE554). A major part of my team's project was gesture acceleartion so I designed a hardware-accelerated image downscaling pipeline that was... a really doozy. Uses a parameterizable (try saying that few times fast), self-pipelining adder tree to interpolate and downscale live frames.

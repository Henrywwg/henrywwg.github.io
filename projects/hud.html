For a while I've wanted to make a HUD style wearable display, but have never really had a use for one. So when I thought about making a Protogen head (which is a sort of robot-dog if you will) I finally had a use for it. Because 'Protogen' is annoying to type I'll just abridge it as 'proot'.

The biggest problem with the proot helmet is the limited visibility from having a visor in front of your face that is packed full of electronics - it's hot and confined.

So the ability to mount some cameras on the outside, and then be able to stream that into the helmet to give you more visibility seems reallly nice. But the biggest problem remains that theres no space in that darn helmet and has been the main restriction with this project. My solution so far has been to take an off the shelf phone 'VR' headset, slap in a 5.5" display and then hook that up to a Raspberry Pi to do the actual image processing. Unfortunately, the whole HUD assembly is too large to fit into the helmet/head so some resizing and reshaping will be necessary to actually get everything working.
